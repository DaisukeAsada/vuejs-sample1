# vuejs-sample1
Vue.jsで作成したサンプルプログラムです。日本各地の天気予報を表示します。


# Description
https://home.openweathermap.org/ で公開されているAPIを利用しています。


# Usage

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
